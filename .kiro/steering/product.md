# Product Overview

AI配信チャットアプリ「Movie Tuber」- AIとのチャットをリアルタイムで配信するWebアプリケーション

## Core Capabilities

- **リアルタイムAIチャット**: ユーザー入力に対してLLMが応答を生成
- **動画生成連携**: AIの応答に合わせてアバター動画を動的に生成・再生
- **配信向けUI**: ループ動画背景、画面モード切替、コントロールボタン
- **外部コメント連携**: わんコメ（OneComme）を通じた配信コメント取り込み

## Target Use Cases

- AIアバターを使った配信・ストリーミング
- インタラクティブなAIキャラクターとの対話体験
- 視聴者コメントに反応するAI配信

## Value Proposition

- チャット→LLM応答→動画生成→再生の一連フローを統合
- 配信画面として使える全画面動画UI
- バックグラウンド動画とアクション動画のシームレスな切り替え
